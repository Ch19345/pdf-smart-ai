> 版本：v1.0 | 日期：2026-03-28 | 状态：初稿

## 1. 产品概述

### 1.1 产品定位

面向海外用户的**极简 PDF AI 助手**，让用户快速获取 PDF 文档的关键信息。

### 1.2 核心价值主张

| 用户痛点 | 我们的解决方案 |
|----------|----------------|
| 阅读长 PDF 太耗时 | AI 几秒生成摘要 |
| 找不到关键信息 | 对话式问答，精确定位 |
| 不想花钱先试试 | 免费版 3 次/天 |

### 1.3 目标市场

- **主要**：美国、英国、加拿大、澳大利亚
- **语言**：英文界面
- **用户画像**：
  - 学生（论文、资料）
  - 职场人（报告、合同）
  - 研究者（论文、文献）

## 3. 用户流程

### 3.1 核心用户路径

```
┌─────────────────────────────────────────────────────────────────┐
│                         用户流程图                               │
└─────────────────────────────────────────────────────────────────┘

[访问首页] → [上传PDF] → [注册/登录] → [AI处理] → [结果展示]
                ↓
          [免费版限制] → [引导升级Pro]
```

### 3.2 详细步骤

#### 步骤 1：首页

- 展示极简上传区域
- 3 个核心卖点
- Sign in / Sign up 按钮

#### 步骤 2：上传 PDF

- 拖拽或点击上传
- 前端验证：PDF 格式、大小限制
- 显示上传进度

#### 步骤 3：用户认证

- 未登录：弹出注册/登录弹窗
- 支持：Email + Google OAuth

#### 步骤 4：AI 处理

- 解析 PDF 文本
- 生成摘要
- 准备对话

#### 步骤 5：结果展示

- 左侧：PDF 预览
- 右侧：AI 对话界面

#### 步骤 6：导出/升级

- 免费用户：引导升级
- 付费用户：导出功能

## 5. 技术架构

### 5.1 技术栈

| 层级 | 技术选择 |
|------|----------|
| 部署平台 | Cloudflare Workers |
| 前端 | HTML + Tailwind CSS |
| 后端 | Workers (Edge Functions) |
| PDF 解析 | pdf-lib / pdfjs |
| AI | OpenAI API (GPT-4) |
| 支付 | Stripe Checkout |
| 存储 | 内存处理（无持久化）|

### 5.2 API 设计

| 端点 | 方法 | 说明 |
|------|------|------|
| `/` | GET | 首页 |
| `/workspace` | GET | 工作区 |
| `/pricing` | GET | 定价页 |
| `/api/upload` | POST | 上传 PDF |
| `/api/chat` | POST | AI 对话 |
| `/api/checkout` | POST | Stripe 跳转 |
| `/success` | GET | 支付成功 |

### 5.3 数据流

```
用户上传 PDF
    ↓
Cloudflare Workers 接收
    ↓
内存中解析 (pdf-lib)
    ↓
提取文本 → 发送给 OpenAI
    ↓
返回结果 → 前端展示
```

### 5.4 限制说明

- **请求大小**：Cloudflare Workers 默认 1MB，可申请到 15MB
- **免费版限制**：10MB，足够大多数文档
- **大文件**：Pro 版可考虑 Cloudflare Stream 或 R2

## 6. 支付流程

### 6.1 Stripe 集成

```
[用户点击 Upgrade]
    ↓
调用 /api/checkout
    ↓
后端创建 Stripe Checkout Session
    ↓
返回 Stripe URL
    ↓
用户跳转 Stripe 支付
    ↓
支付成功 → 跳转 /success
    ↓
Stripe Webhook 更新用户订阅状态
```

### 6.2 订阅管理

用户可通过 Customer Portal：
- 查看账单
- 更新支付方式
- 取消订阅
